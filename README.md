# Учебная задача: Анализ сайта «СберАвтоподписка»

Данная учебная задача включала в себя следующие важные этапы:
1. Поиск целевых действий пользователей. Из 245 исходных типов событий были выбраны 18 наиболее подходящих для создания целевой переменной (успешности сессии). 
2. Создание целевой переменной для задачи классификации.
3. Анализ исходного датасета, необходимый для выявления механизмов предобработки данных.
4. Создание пайплайна предобработки. В том числе написание кастомных классов предобработки, которые можно найти в файле src/transformers.py
5. Разведочный анализ (EDA).
6. Подбор параметров для двух моделей классификации на обучающей выборке.
```text
     Модель: LightGBM
==================================================
    Лучшие параметры: {'learning_rate': 0.1, 'n_estimators': 200}
    ROC-AUC на тестовой выборке: 0.7151
```   
7. Обучение полного пайплайна с лучшей моделью на всем наборе данных и сохранение для дальнейшего инференса.
8. Создание сервиса нна базе Flask, который принимает характеристики визита пользователя на сайт и предсказывает, совершит ли он целевое действие (ориентировочно: покупка, подписка или отправка заявки и др.).

## 🚀 Инструкция по запуску

### 1. Установка зависимостей
Убедитесь, что у вас установлены все необходимые библиотеки из requirements.txt

### 2. Запуск веб-сервиса
Перейдите в корень проекта (папка `app/`) и запустите Flask-сервер:
```bash
python app.py
```
Сервер поднимется локально по адресу: `http://localhost:5000`.

### 3. Запуск интеграционных тестов
Для проверки работоспособности модели на реальных данных, не выключая сервер, откройте второй терминал и запустите скрипт тестирования:
```bash
python tests/test_prediction.py
```
Скрипт выберет случайную строку из тестового набора, отправит JSON-запрос в API, и замерит время отклика.

## ⏱️ Производительность и метрики
* **ROC-AUC модели на тестовой выборке:** 0.7151
* **Время отклика (Latency):** Сервис полностью укладывается в техническое задание (менее 3000 мс), демонстрируя среднее время обработки запроса в районе **2000 мс** на локальной машине.
